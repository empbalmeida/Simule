Banco de Dados Compartilhado
Qualquer pessoa com o link vai ver e salvar no mesmo banco de dados. Configure com o Supabase (gratuito).
Passo 1 — Criar projeto gratuito
1. Acesse supabase.com e crie uma conta gratuita
2. Crie um novo projeto (guarde a senha)
3. Vá em Project Settings → API
4. Copie a Project URL e a chave anon public
Passo 2 — Criar a tabela
No Supabase, vá em SQL Editor e execute:

create table simulacoes (
  id text primary key,
  dados jsonb not null,
  created_at timestamptz default now()
);
alter table simulacoes enable row level security;
create policy "public" on simulacoes for all using (true) with check (true);
